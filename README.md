A simple shopping web application built using JSP, Servlets, and MySQL.  
Users can log in, browse products, add items to cart, remove selected items, and purchase all products.

## Features
- Login & Logout (Session-based)
- Product pages: Mobiles, Laptops, Watches
- Add to Cart / Remove Selected Items
- Purchase items & show purchased list
- Fully database-driven (JDBC + MySQL)

## Tech Stack
- Java (JSP + Servlets)
- Tomcat Server
- MySQL Database
- HTML & CSS
